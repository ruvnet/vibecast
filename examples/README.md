# AgentDB v3 Examples - Complete Capability Showcase

> 20 examples from practical to exotic, demonstrating every major capability of `agentdb@alpha` (3.0.0-alpha.3)

## Quick Start

```bash
# Run a single example
node examples/practical/01-unified-agentdb.js

# Run all examples
node examples/run-all.js
```
