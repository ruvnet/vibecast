# Vibecast - Nuclear Power Plant Management Simulation

Weekly Vibecast Live coding sessions with rUv. Check branches for each week.

## Current Project: Next-Generation Nuclear Power Plant Simulation

A high-fidelity nuclear power plant management system featuring:
- Industrial Control Systems (ICS) for reactor operations
- Complete supply chain management
- Human resources and workforce management
- Business operations and financial modeling
- 5 federated AI agents powered by E2B sandboxes
- Real-time analytics with ruvector and RuvLLM (SIMD)

### Quick Start

```bash
npm install
export E2B_API_KEY=your_key_here
npm start
```

See detailed documentation in the project files.
