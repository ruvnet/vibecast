### 1. Biometric Sensor System
**File:** `src/biodata/sensors.ts`

```typescript
class BiometricSensors {
  - Heart Rate Variability (HRV) - stress/relaxation
  - EEG bands (alpha, beta, theta, gamma) - mental states
  - Galvanic Skin Response (GSR) - emotional arousal
  - Respiratory rate - calm vs agitated
  - Simulate realistic biometric streams for demo
  - Parse real sensor data (prepare for actual devices)
}
```

Types of data:
- **HRV**: Time between heartbeats (high = relaxed, low = stressed)
- **EEG**: Brainwave frequencies
  - Alpha (8-13Hz): Relaxed awareness
  - Beta (13-30Hz): Active thinking
  - Theta (4-8Hz): Meditation, creativity
  - Gamma (30-100Hz): Peak focus
- **GSR**: Skin conductance (arousal, emotion)
